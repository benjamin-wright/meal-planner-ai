## The Basics

- Target environment: mobile
- Typescript for type safety: (reference: https://www.typescriptlang.org/docs/handbook/)
- React version 19.2.0 (reference: https://react.dev/reference/react)
- Storybook for visual prototyping (reference: https://https://storybook.js.org/docs)
- Playwright for end-to-end testing
- PWA (Progressive Web App): Platform-agnostic and entirely offline, to support use in environments with limited or no reception
- IndexedDB persistence: local, performant and ubiquitous (reference: https://developer.mozilla.org/en-US/docs/Web/API/IndexedDB_API)
- Vite for build: popular and fast (reference: https://vite.dev/guide/)
